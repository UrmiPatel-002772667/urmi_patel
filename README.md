## 🔧 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/UrmiPatel-002772667/urmi_patel.git
   ```
2. Navigate to the project directory:
   ```bash
   cd urmi_patel
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
